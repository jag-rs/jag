# jag-text

Text layout, shaping, and editing utilities for [jag-draw](https://crates.io/crates/jag-draw).

Part of the [jag](https://crates.io/crates/jag) toolkit. This is an internal crate — most users should depend on `jag` or `jag-draw` instead.

## Features

- Font loading and metrics
- Text shaping via harfrust
- Unicode segmentation, bidi, and line breaking
- Undo/redo for text editing
